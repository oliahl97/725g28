# 725G28
Databases and Data Modelling

Work from the 725G28 Databases and Data Modelling course at Linköping University.  
The course focused on how to design and use relational databases, from modelling to implementation and querying.

## What I worked with
- database design: entities, relationships, constraints, and normalization
- ER modelling (ER diagrams) and translating models into a relational schema
- implementing tables, keys, indexes, and integrity rules
